Quelques erreurs...

Déjà dans les niveaux Saturn, il y a un boss dans l'un d'eux... 

Ensuite, le jeu avec Maria est excellent car plutot basé sur la technique. C'est un défi de jouer avec elle qui prend tout son ampleur contre un Galamoth. Idem pour Richter (dans une moindre mesure pour lui). Ils sont là pour le scoring, pas pour la storyline.

Ensuite le 60hz ne prévaut que pour la version PS1 PAL. Les version PS1 Jap et Saturn sont DEJA 60 hz...

Le mode 16/9 est une blague aussi sur cette version XBLA. L'option optimisation des graphismes enlaidit le jeu plus qu'elle ne l'embellit... Les FMV ont disparu aussi. Les musiques ne sont pas de super qualité du fait de la compression j'ai trouvé, c'est flagrant sur certains thèmes.

Bref, les meilleures versions, ce sont clairement les versions Saturn et PSP (le VOSTFR, ça jette, les voix anglaises sont nazes). Le seul attrait majeur de cette version XBLA, c'est son prix, rien d'autre. La langue Française pour les textes, c'est pas mal mais le Japonais ne bloque strictement rien (il est en Francais sur PSP aussi).

Autre erreur, le jeu a un style gothique et non pas manga.

Pour ceux que ça intéresse, j'en ai fait un gros comparatif sur mon blog sur Gamekult. Ce n'est pas une pub mais ce jeu, je l'ai entièrement dépoussiéré ^^

Le jeu XBLA est pour moi la moins bonne version de ce fantastique jeu.
