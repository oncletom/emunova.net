Boîte du jeu de danse Para Para Paradise, uniquement disponible sur l'archipel nippon. Cette boîte contient un ensemble de 5 plots à détecteurs infrarouges, le tout à relier via un câble USB (fourni) sur sa Playstation 2\. Bien que japonais, ce matériel fonctionne sur n'importe quelle console (européenne ou américaine), seul le jeu lui-même étant zoné.
