Les gens qui ont trouvé le jeu trop facile, même en mode Hard ne se sont jamais penchés sur le fait que l'on pouvait se handicaper grâce au code fourni après avoir justement terminé ce mode. Non, Kirby n'est pas aussi simple que cela : la preuve, vous pouvez doser votre difficulté en vous enlevant votre nombre maximum de points de vie. Le jeu est graphiquement bien fait, les musiques sont sympas et le tout est plein d'humour. Un jeu qui vaut le coup d'être essayé.
