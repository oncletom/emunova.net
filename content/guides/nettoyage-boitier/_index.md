# Des boitiers presque neufs :

Le marché de l'occasion des jeux vidéo permet de pouvoir rejouer à de nombreux hits sur de nombreux supports et c'est ainsi que vous pouvez vous constituer une collection de jeux pour presque rien.

L'inconvénient principal est que les magasins étiquettent les jeux d'occasions directement sur la boite d'origine. Dans certains cas, il n'y a que le prix, dans d'autres vous avez l'étiquette de pub du magasin. Il faut reconnaître que cela peut être disgracieux et c'est pour cela que nous allons vous proposez ce petit guide afin d'éliminer les intrus.
