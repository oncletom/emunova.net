Malgré les défauts cités dans le test j'ai trouvé ce jeu est assez sympa à faire. 
La sensation de vitesse est vraiment impressionnante pour l'époque et le moindre faux pas se paie cash sur le chrono du moins sur les deux premiers circuits. Sur le troisième, la méthode bourrin (taper les murs de coté dans les virages) est assez payante.

Je peux comprendre que le jeu soit une déception pour certains : la jouabilité n'est pas un exemple de finesse, les chronos se battent trop facilement, le jeu est assez facile et rapide à finir mais surtout une fois terminer on a pas forcément envie de revenir dessus.
