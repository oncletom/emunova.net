Le Transfer Pack permettait de placer une cartouche Game Boy dans une manette Nintendo 64 afin d'en extraire les sauvegardes.

Cet accessoire fût notamment utilisé pour Pokémon, histoire de pouvoir jouer aux Pokémon Stadium avec son bestiaire de Pokémon débloqué dans les versions Game Boy.
